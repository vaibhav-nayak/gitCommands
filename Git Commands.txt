1. what is git :  git is open source version control tool , i.e to track , switch between various version of software 
2. git --version: To Check Version of git
3. git init  : to initialize git repository(folder)
4. git config --global user.name "user_name" : to initialize user profile
5. git config --global user.email "email_nam": to initialize user email
6. git status: to check current status of git repo
7. git add. : will add all files and folder in stagging area(Area of tracking)
8. git commit -m "message here": will apply and move changes to local repository and unique hash is generated corresponding to each commit
9. git log : will show all the commit
